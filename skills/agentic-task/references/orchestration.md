# Orchestration Notes

## Delegation checklist

- [ ] Define the deliverable before delegating
- [ ] Give each subagent a narrow, non-overlapping job when possible
- [ ] Pass only the artifacts needed for that job
- [ ] Require evidence, file paths, and explicit uncertainty
- [ ] Keep synthesis separate from collection

## Useful reviewer combinations

- Broad code review: `correctness-reviewer` + `security-reviewer` + `performance-reviewer` + `simplicity-reviewer`
- Spec review: `spec-reviewer` + `security-reviewer` + `performance-reviewer` + `simplicity-reviewer`
- Implementation prep: `scout` -> `planner`
